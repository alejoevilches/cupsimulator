Bienvenido al simulador de torneos

Esta aplicación te permitirá simular cualquier tipo
de torneo bajo la estructura básica mas conocida
y difundida en el mundo (32 equipos, 8 grupos,
fase eliminatoria). Estas son las instrucciones para
su uso:

  1. Debes ingresar todos los equipos participantes en cada grupo.
  2. Ingresá el resultado de cada partido del grupo. El sistema automaticamente calculará todas las variables
  3. Una vez que están definidos todos los cruces eliminatorios podrás seleccionar quien será el vencedor de cada llave. Así hasta llegar hasta el final.
  4. En cualquier momento podrás resetear el simulador para volver a arrancar, asi como también volver a ver estas instrucciones.
